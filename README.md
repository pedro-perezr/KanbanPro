KANBANPRO

App de gestion de tareas estilo Kanban.

## Para usarlo debe escribir en consola:
npm install
node app.js
Abrir en `http://localhost:3000`

## Funcionalidades
- Crear, renombrar y eliminar tableros
- Crear, renombrar y eliminar columnas
- Agregar y eliminar tarjetas con color personalizado
- Modo oscuro
- Datos persistentes en `data.json`
